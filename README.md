# Zomato Restaurant Data Analysis

Python-based Exploratory Data Analysis (EDA) project built using Pandas, NumPy, Matplotlib, and Seaborn. The project focuses on data cleaning, visualization, and extracting meaningful insights from the Zomato restaurant dataset.

## Features

* Data Cleaning and Preprocessing
* Restaurant Rating Analysis
* City-wise Restaurant Analysis
* Cuisine Analysis
* Online Delivery Analysis
* Price Range Analysis
* Votes vs Rating Analysis
* Correlation Heatmap
* Data Visualization using Seaborn and Matplotlib

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Visualizations

* Bar Plot
* Count Plot
* Scatter Plot
* Pie Chart
* Heatmap

## Project Outcome

This project demonstrates practical Python programming, data manipulation, exploratory data analysis (EDA), and visualization skills by generating meaningful business insights from restaurant data.
